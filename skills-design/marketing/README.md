# Marketing / 营销类 Skills（顾问 / 市场 / 保险）

这里沉淀且慢营销规范与可复用模板，优先覆盖：顾问内容、市场活动、保险业务。

## 使用方式（推荐）
1. 先看本页目录 → 找到对应场景（顾问/市场/保险）
2. 复制对应模板/规范到你的项目里使用
3. 产出新的经验：新增一个 `.md` 文件并补到目录

## 文件命名规则（不增加文件夹，通过前缀区分）
- `advisor_*.md`：顾问（投顾内容、投教、陪伴式表达、顾问话术）
- `market_*.md`：市场（活动、投放、H5、海报、短视频）
- `insurance_*.md`：保险（产品卖点、条款表达、合规提示、投放规范）

> 示例：`market_h5-structure.md` / `advisor_script-template.md` / `insurance_poster-spec.md`

## 常用检查清单（快速自检）
- 品牌一致性：色彩/字体/Logo露出/卡片样式统一
- 文案正确：金额、期限、风险提示、禁用词
- 合规与敏感：不夸大收益、不绝对化表述
- 渠道适配：比例/安全区/封面标题可读
