# Fonts（营销用）

> 本仓库为 Public：默认不上传字体文件（ttf/otf），仅记录字体清单、使用规则与获取方式，避免授权风险。

## 字体清单
- 中文：Alibaba PuHuiTi（阿里巴巴普惠体）
- 数字/英文：MiSans（小米字体）

## 使用规则（推荐）
- 标题（中文）：Alibaba PuHuiTi Medium / Bold
- 正文（中文）：Alibaba PuHuiTi Regular
- 数字（金额/收益率/百分比）：MiSans Medium（关键数字）/ Regular（普通数字）
- 小字注释：Alibaba PuHuiTi Regular（字号更小、行高更松）

## 场景建议
- 数据看板/收益率：数字优先 MiSans，保证对齐与可读性
- 海报/H5：标题中文用普惠体，关键数字用 MiSans 拉开层级
