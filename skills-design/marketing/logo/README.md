# Logo（营销用）

本目录用于存放且慢品牌 Logo 的营销投放版本（含透明底 PNG / 矢量 SVG 等）。

## 推荐使用
- **浅色背景（白/浅灰）**：优先使用 `Color` 或 `Black`
- **深色背景（深蓝/黑）**：使用 `White`
- **小尺寸/角标/头像位**：使用 `Icon`（仅图形符号）

## 文件命名建议
`Qieman_Logo_<Variant>_<Color>_<Layout>.<ext>`

- Variant：`Lockup`（含文字）/ `Icon`（仅图形）
- Color：`Color` / `Black` / `White`
- Layout：`Horizontal`（横版）/ `Stacked`（竖版，可选）

示例：
- `Qieman_Logo_Lockup_Color_Horizontal.png`
- `Qieman_Logo_Lockup_White_Horizontal.png`
- `Qieman_Logo_Icon_Black.png`

## 使用规范（必看）
- 禁止拉伸变形、加描边/阴影、随意改色
- 保持安全留白（四周至少留出 Logo 高度的 1/4）
- 避免放在高频复杂纹理背景上，必要时加纯色底或模糊遮罩
