# Logo｜深色/暗色背景用

在深灰、深蓝、黑色等**深色/暗色背景**上使用的 Logo 切图。

## 已纳入文件

| 文件名 | 说明 |
|--------|------|
| `且慢logo-white.png` | 且慢 Logo（白标 + 白字） |
| `且慢logo-Graphic-white.png` | 且慢 Logo 图形版（白） |
| `logo-slogan-white.png` | 且慢 + 口号「安放财富 · 静待花开」（白） |
| `盈米且慢logo-white.png` | 盈米基金 + 且慢 组合（白） |
| `盈米logo-white.png` | 盈米基金 独立 Logo（白） |
| `Group 427319881.png`、`Group 427319884.png` | 设计稿导出备用 |

## 调用方式

- **且慢 Logo**：`<img src="且慢营销规范/logo/深色背景/且慢logo-white.png" alt="且慢" />`
- **且慢+口号**：`<img src="且慢营销规范/logo/深色背景/logo-slogan-white.png" alt="且慢 安放财富·静待花开" />`
- **盈米+且慢**：`<img src="且慢营销规范/logo/深色背景/盈米且慢logo-white.png" alt="盈米基金 且慢" />`
- **CSS 变量**：`--logo-qieman-dark: url('./且慢营销规范/logo/深色背景/且慢logo-white.png');`
