# 且慢营销配色规范（Marketing Color Skills）

## 0. 概述

本技能在且慢 UI 品牌主色基础上，扩展**常规营销基调（蓝/紫）+ 新春系列（红金）+ 高净值鎏金蓝 + 通用中性色**，用于统一且慢在不同营销场景下的色彩风格。

**强制约定（与且慢品牌设计规范一致）**：

- **营销按钮与 CTA**：统一遵循且慢品牌设计规范，**主按钮必须使用品牌蓝**（背景 `#1B88EE`、文字 `#FFFFFF`、圆角 8px、内边距 12px 24px），不随营销基调（红/金/紫）改变；次按钮/文字按钮为文字 `#1B88EE` 或轻底 `rgba(27,136,238,0.1)`。详见《且慢营销设计规范》5.4 节及《且慢 UI 设计规范》按钮章节。

**关键词**：且慢营销、配色规范、品牌蓝、新春红金、科技紫、高净值、蓝色按钮、CTA

**适用场景**：活动 H5/落地页、运营位与 Banner、战报与数据长图、高净值/私募页、AI/创新功能宣发；**所有场景下主 CTA 均为品牌蓝按钮**。

---

## 1. 选色决策总览

1. **先选基调**：日常/品牌 → 品牌蓝 A；深度/高端 → 稳重深蓝 B；AI/创新 → 科技紫 C；春节/节庆 → 新春红金；高净值 → 深蓝 B + 克制金色点睛。
2. **再选层级**：600 主按钮/主标题点睛（**主 CTA 仍用品牌蓝 #1B88EE**）、500 hover/强调、100–200 背景氛围。
3. **通用降噪**：用 Neutral + Solid 做底色与正文，保证可读性。

## 6. 高净值｜鎏金蓝（组合思路）

**定位**：高净值、私募、高端服务。在稳重深蓝 B 上叠加克制金色点睛。

- 背景：B 基调 700–900（如 `#061D32`、`#0A3257`）
- 文字：Neutral 深灰（`#2C343A` / `#101418`）
- 金色：Gold 400–600 小面积描边/图标/标题点（如 `#FFCC66`、`#FFAA00`）
- **主 CTA**：**品牌蓝 `#1B88EE`**，与且慢品牌设计规范一致。

## 8. 营销按钮与 CTA（强制遵循品牌规范）

无论使用哪一套营销配色基调（品牌蓝 A、深蓝 B、科技紫 C、新春红金、高净值鎏金蓝），**营销按钮均按且慢品牌设计规范执行**：

| 元素 | 规范 | 说明 |
|------|------|------|
| **主按钮** | 背景 `#1B88EE`，文字 `#FFFFFF`，圆角 8px，内边距 12px 24px | 一屏最多一个主 CTA |
| **次按钮/文字按钮** | 文字 `#1B88EE`，可选轻底 `rgba(27,136,238,0.1)`（hover） | 不随主题改为红/金/紫 |
| **禁用** | 背景/文字 `#CCCCCC` | 与 UI 规范一致 |

**口诀**：基调管背景和氛围，**按钮永远品牌蓝**。

## 10. 使用口诀

- **选主题**：日常→A，深度/高端→B，AI/创新→C，春节→红金，高净值→B+金。
- **定主色档位**：主标题/主视觉可选对应基调 600；**主按钮/CTA 固定品牌蓝 `#1B88EE`**。
- **控数量**：一页一个主色基调 + Neutral/Solid；红/金/紫不用于主 CTA 按钮。
- **先清晰再好看**：优先保证文字与数据可读性与层级，再考虑渐变与装饰。
