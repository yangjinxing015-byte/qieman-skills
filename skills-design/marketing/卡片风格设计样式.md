# 且慢 H5 卡片风格样式设计规范（Card-Style Design）

> 本规范基于 H5 营销页视觉与板块框架，定义**卡片风格**的整体布局、模块结构、组件形态与复用方式。适用于互认基金、投研专题、产品介绍等场景，与《且慢营销设计规范》《且慢营销配色规范》《营销头部氛围图设计 SKILL》配合使用。

---

## 1. 概述与适用范围

### 1.1 目标

- **模块化**：页面按「Hero 卡 + 内容卡」积木拼装，支持快速增删板块。
- **可读性**：卡片分割清晰，信息层级明确，降低理解成本。
- **一致性**：与且慢品牌视觉同源，主 CTA 统一品牌蓝。

### 1.2 适用范围

- 互认基金、投研专题、产品介绍等 H5 营销页
- 落地页、长图页中的卡片式内容区
- 需支持「不同人物/插画 + 不同背景色」的 Hero 变体

### 1.3 设计原则

- **卡片为主**：主内容以白/浅底卡片呈现，模块间留白充足。
- **结构统一**：标题 → 说明 → 子卡片/数据 → CTA 的固定模式可复用。
- **人物/插画可替换**：Hero 区支持多套人物或概念插画，背景色按素材主色适配（见《营销头部氛围图设计 SKILL》）。

## 2. 页面结构框架

### 2.1 三段式结构（自上而下）

| 区域 | 用途 | 说明 |
|------|------|------|
| **顶部 Hero 卡** | 主标题 + 简介 + 人物/概念插画 | 背景色可随插画主色变化（浅粉/橙/蓝/黄等柔和色）；左文右图，文字与插画顶对齐；投研/策略类可选**深色 Hero**（稳重深蓝 B 基调 `#061D32`～`#125EA5`，白字） |
| **中部内容卡** | 投资目标、配置组合、类型说明等 | 多个独立卡片纵向堆叠；每卡含标题、正文、子卡片或数据可视化 |
| **底部扩展区** | 免责声明、详细说明、风险提示 | 小字号、次级色，不抢夺主内容 |

### 2.2 典型布局示意

```
┌─────────────────────────────────┐
│  Hero 卡（主标题 + 简介 + 插画）  │  ← 背景：浅粉/橙/蓝/黄等
├─────────────────────────────────┤
│  内容卡 1（投资目标）             │
│  ├ 子卡 / 标签 / 数据            │
│  └ [立即查看] 按钮               │
├─────────────────────────────────┤
│  内容卡 2（配置组合）             │
│  ├ 饼图 / 占比                   │
│  └ [了解详情] 按钮               │
├─────────────────────────────────┤
│  内容卡 3（稳健型 / 进取型 等）   │
│  ├ 投资期限、预期收益            │
│  └ [查看详情] 按钮               │
├─────────────────────────────────┤
│  底部：免责声明 / 风险提示        │
└─────────────────────────────────┘
```

## 3. 卡片视觉规范

### 3.1 基础卡片（内容卡）

| 属性 | 取值 | 说明 |
|------|------|------|
| **背景** | `#FFFFFF` | 纯白底，与页面底 `#F9FAFB` 形成对比 |
| **边框** | 无 | **卡片不使用边线** |
| **阴影** | 无 | **卡片不使用投影**，以留白区分层次 |
| **圆角** | 8px（小）/ 12px（中） | 与《且慢营销设计规范》3.3 一致 |
| **内边距** | 12px～16px | 卡片内留白 |
| **模块间距** | 16px | 卡片与卡片之间 |

**页面底色**：`#F9FAFB`（浅灰），衬托白卡。

### 3.2 Hero 卡（首屏区）

| 属性 | 取值 | 说明 |
|------|------|------|
| **背景** | 浅色渐变（配色规范 100–200 档）或**深色 Hero** | 浅粉、浅橙、浅蓝、浅黄等；投研/策略类可用稳重深蓝 B（`#061D32`～`#125EA5`），白字 |
| **主标题** | 26pt 加粗、`#333333`（浅色）/ `#FFFFFF`（深色） | 与《营销头部氛围图设计 SKILL》一致 |
| **简介文案** | 16pt 常规、`#606060` / 浅色半透明（深色 Hero） | 1–3 行 |
| **插画/人物** | 右侧，图 1 或图 2 比例 | 见《营销头部氛围图设计 SKILL》2.5 |

### 3.3 子卡片（嵌套在内容卡内）

| 属性 | 取值 | 说明 |
|------|------|------|
| **背景** | `#F0F6FF` / `#F9FAFB` | 浅蓝信息底或页面底，与主卡区分 |
| **边框** | 无 | 与主卡一致，不设边线 |
| **圆角** | 8px | 与主卡一致 |
| **内边距** | 10px～12px | 略小于主卡 |

### 3.4 彩色顶条标题卡（Header Banner Card）

卡片顶部设**圆角顶条**，可略超出卡片顶缘，形成视觉锚点：

| 属性 | 取值 | 说明 |
|------|------|------|
| **顶条背景** | 浅蓝 `#DBEBFF` / 浅橙 `#FDDBD3` / 浅紫 `#E7DBFF` | 按主题或语义选择，与配色规范 100–200 档一致 |
| **顶条圆角** | 8px 或 12px | 与卡片圆角协调 |
| **标题文字** | 16pt～19pt 加粗、居中 | 置于顶条内 |
| **顶条与卡片关系** | 可微露出卡片顶部或完全内嵌 | 按设计稿选用 |

### 3.5 标签 / 关键词胶囊

| 属性 | 取值 | 说明 |
|------|------|------|
| **背景** | 淡紫 `#E7DBFF` / 淡绿 `#D4EDDA` / 淡黄 `#FFF3CD` | 按语义区分 |
| **圆角** | 999px（全圆角） | 胶囊形 |
| **字号** | 12px～14px | 小标签 |

## 4. 内容卡组件模板

### 4.1 卡片形态变体（参考图）

| 形态 | 结构 | 适用场景 |
|------|------|----------|
| **顶条标题卡** | 彩色顶条（蓝/橙/紫）+ 正文 + 底部图位 | 投资目标、产品介绍、通用说明 |
| **顶条 + 图标装饰卡** | 彩色顶条 + 两侧装饰图标 + 正文 + 底部图位 | 强调型、仪式感较强的模块 |
| **数据可视化卡** | 标题（白底左对齐）+ 环形图 + 彩色图例（圆点+占比+描述句）+ 底部说明段落 | 配置组合、资产占比、策略说明 |
| **数据对比统计卡** | 浅黄底区块 + 白底数据格（大数字+小标签）+ 柱状图 | 累计收益率、年化、回撤等指标对比 |
| **柱状图卡片** | 多种形态：业绩走势、净值对比、方案推荐、资产表现等 | 见 4.7 |
| **用户对比图卡** | 左右双柱（3D 圆柱）+ 气泡数据 + VS 分隔 + 底部结论句 | 清仓用户 vs 收益 Top10% 等对比，见 4.7.8 |
| **年度总结卡** | 主指标大字号 + 麦穗装饰 + 多行白底数据格 + 成就标签（图标+次数） | 年度收益率、年度概览、年度数据、策略年度账单，见 4.3.9 |
| **风险警示区块** | 浅粉底 + 红色图标 + 警示文案 | 风险事件、黑天鹅提醒 |
| **图标组卡** | 圆形深蓝图标横排 + 下方标签 | 人生目标、风险承受、投资期限等概念 |
| **分区标题卡** | 蓝竖条 + 标题 + 正文 + 子卡 | 传统分区结构，见 4.2 |

### 4.3.1 数据对比统计卡（指标块 + 柱状图）

用于展示累计收益率、年化收益率、最高回撤等**多指标并排对比**，配合柱状图呈现趋势。

```
┌─────────────────────────────────┐
│ 01 市场是动态的                  │  ← 主标题：编号 + 标题，深蓝加粗
│ 实现收益目标的过程也是动态的      │  ← 副标题：深灰，略小
├─────────────────────────────────┤
│ ┌─ 浅黄底区块（细黄边）─────────┐ │  ← 背景 #FFF7E5 / #FFFAEF，边框 #FFDD99
│ │ ┌──────┐┌──────┐┌──────┐    │ │     内为 3 个白底数据格，细灰边
│ │ │ 282% ││8.74% ││68.86%│    │ │     大数字 16–24pt 加粗，小标签 12–14pt
│ │ │累计收││年化收││最高  │    │ │
│ │ │益率  ││益率  ││回撤  │    │ │
│ │ └──────┘└──────┘└──────┘    │ │
│ │ ▓▓ ▓▓ ▓▓ ▓▓ ▓▓ (柱状图)      │ │  ← 深蓝 #125EA5 与金色 #FFCC66 交替
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| **区块外框** | 浅黄底 `#FFF7E5` 或 `#FFFAEF`，细边框 `#FFDD99` 或 `#FFCC66` |
| **数据格** | 白底、细灰边、大数字 16–24pt 加粗、小标签 12–14pt 次级色 |
| **柱状图** | 深蓝 `#125EA5` 与金色 `#FFCC66` 交替，与《且慢营销配色规范》B 基调 + Gold 一致 |

### 4.3.2 风险警示区块

用于风险事件、黑天鹅提醒等警示内容，浅粉底 + 红色图标强化感知。

```
┌─────────────────────────────────┐
│ ● 2018年 中美贸易摩擦 ...        │  ← 红色三角/星号/日历图标 + 警示文案
│ ● 2020年 全球新冠疫情 ...        │     背景 #FEEDE9 / #FEEBE7
│ ● 2022年 美联储激进加息 ...      │     图标与重点文字红色 #FA420A
└─────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| **背景** | 浅粉 `#FEEDE9` 或 `#FEEBE7`（Red 100） |
| **图标** | 红色三角警告、星号、日历等，`#FA420A` |
| **文案** | 深灰主文，关键年份/事件可红色强调 |

### 4.3.9 年度总结卡（年度收益率 / 年度概览 / 年度数据）

用于**年度账单、策略年度总结、年度数据概览**等场景，突出主指标（如年度收益率），辅以多行数据格与成就标签。

```
┌─────────────────────────────────┐
│         年度收益率                │  ← 标题居中，深灰 #4A4A4A
│                                 │
│   🌾    +10.22%    🌾           │  ← 主指标居中，两侧麦穗左右对称摆放
│   (左)              (右)         │     左麦穗：左对齐于「+」左侧，右麦穗：右对齐于「%」右侧
│                                 │     麦穗高度 ≈ 主指标字高的 60%～70%，宽度窄于高度
│                                 │     麦穗与数字间留白 12～16px，不贴数字
│                                 │     左右镜像对称；渐变色粉橙 #F0C0B0 → 浅蓝 #B0C0E0
│                                 │     素材路径：且慢营销规范/麦穗/麦穗装饰-年度收益率.png
│                                 │     主指标橙红 #FF6633
│                                 │
│  ┌────────┐┌────────┐┌────────┐  │  ← 第一行：3 个白底数据格，横向排列，无边框
│  │年度最大││盈利用户││平均累计│  │     圆角 8px，留白区分
│  │回撤    ││占比    ││收益率  │  │
│  │-7.22%  ││99.23%  ││5.34%   │  │     负值绿色 #07AD8F，正值/普通值黑色 #333333
│  └────────┘└────────┘└────────┘  │
│                                 │
│  ┌──────────────┐┌──────────────┐│  ← 第二行：2 个较宽成就卡，左图标 + 标题 + 数值
│  │🏆 净值创新高 ││💰 帮用户止盈 ││     图标可用 emoji 或 remixicon
│  │    12次      ││    3次       ││     标题深灰 14pt，数值 16–18pt 加粗
│  └──────────────┘└──────────────┘│
└─────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| **整体** | 白底、无边框、无投影，圆角 8～12px |
| **标题区** | 居中，「年度收益率」「年度概览」等；深灰 #4A4A4A |
| **麦穗装饰** | 主指标两侧左右对称；左麦穗左对齐、右麦穗右对齐；高度≈主指标字高 60%～70%，宽窄于高；与数字间距 12～16px；左右镜像；渐变粉橙 #F0C0B0→浅蓝 #B0C0E0；**素材路径**：`且慢营销规范/麦穗/麦穗装饰-年度收益率.png` |
| **主指标** | 36～38pt 加粗；正收益橙红 #FF6633；负收益绿色 #07AD8F |
| **数据格行一** | 3 个白底格，横向等分；标题 12～14pt 深灰，数值 16～20pt 加粗；负值绿、正值/中性黑 |
| **成就格行二** | 2 个较宽白底格；左图标（🏆/💰或 remixicon）+ 标题 + 次数；圆角 8px |
| **间距** | 数据格之间 10～12px，行间距 14～16px |

**适用场景**：稳稳的幸福年度账单、策略年度总结、年度收益率展示、年度成就概览。

### 4.3.10 麦穗装饰样式（通用）

用于**年度总结、年度数据、年度概览、收益率总结**等场景，作为主指标（如年度收益率）两侧的装饰元素。

| 属性 | 规范 |
|------|------|
| **素材路径** | `且慢营销规范/麦穗/麦穗装饰-年度收益率.png` |
| **摆放** | 主指标左右对称，左麦穗左对齐、右麦穗右对齐；左右镜像 |
| **比例** | 高度 ≈ 主指标字高的 60%～70%，宽度窄于高度 |
| **间距** | 与数字间距 12～16px |
| **配色** | 渐变色粉橙 #F0C0B0 → 浅蓝 #B0C0E0 |
| **HTML 引用** | `src="./且慢营销规范/麦穗/麦穗装饰-年度收益率.png"` |
| **左麦穗** | `transform: scaleX(-1)` 镜像 |
| **右麦穗** | 原图 |

**file:// 协议说明**：本地双击打开时，相对路径可能解析失败。建议使用本地 HTTP 服务（如 `npx serve .` 或 `python3 -m http.server 8888`）访问，或将图片转为 base64 内嵌以保证显示。

### 4.3.3 图标组卡（圆形图标 + 标签）

用于人生目标、风险承受、投资期限等**概念并列**，深蓝圆形图标横排 + 下方标签。

```
┌─────────────────────────────────┐
│      ◯        ◯        ◯        │  ← 3 个圆形深蓝图标（#125EA5 / #1B88EE）
│   人生目标   目标承受  投资期限   │     下方 14pt 标签，深灰
└─────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| **图标** | 圆形、深蓝 `#125EA5` 或品牌蓝 `#1B88EE` |
| **标签** | 14pt、`#606060`，与图标间距 8px |

```
│ │ 子卡标签     │ │ 子卡标签     │ │

```

### 4.5 分区标题（Section Title）

- **左侧竖条**：4px 宽 × 20px 高，圆角 2px，背景 `#1B88EE`
- **标题文字**：19pt 加粗，`#333333`，与竖条间距 8px

**CSS 示例**：

```css
.sec-title {
  display: flex;
  align-items: center;
```

## 7. 实现要点

### 7.1 模块可删减

- 内容卡使用 `data-module` 或 `data-removable` 标记，支持按业务快速隐藏整块
- 子卡数量可 2 个或 3 个，布局用 `grid-template-columns: repeat(auto-fit, minmax(120px, 1fr))` 自适应

### 7.2 响应式

- 容器宽度：`min(343px, 100%)` 或 `min(430px, 100%)`，居中
- 模块间距、圆角、字号与《且慢营销设计规范》保持一致

### 7.3 刷新切换卡片样式（强制）

**卡片样式须支持刷新后更换**。H5 需提供多种卡片形态（顶条标题卡、分区标题卡、顶条+图标装饰卡等），用户**每次刷新**可看到不同样式，增强新鲜感与 A/B 测试能力。

**实现方式**：

1. **URL 参数**：优先用 `?card=<index>` 记录当前样式索引，刷新后 `(index+1) % total` 切换；`history.replaceState` 更新地址栏。
2. **存储备选**：`localStorage` 或 `sessionStorage` 记录 `qm-card-style-index`，当 URL 不可用时回退。
3. **样式切换**：在根元素或内容容器上设置 `data-card-style="banner"` / `"section"` / `"banner-icon"` 等，通过 CSS 选择器 `.content-card[data-card-style="banner"] .card-banner { display: block }` 控制显隐或布局。
4. **缓存穿透**：若卡片内含图片或动态资源，可附加 `?v=Date.now()` 避免缓存导致样式不更新。

**可用形态**：banner（彩色顶条）、section（蓝竖条+编号）、banner-icon（顶条+两侧装饰图标）。**建议支持 3 种形态轮换**，质检时刷新 4 次须确认 3 种形态均能切换展示。
