# Qieman Design Skills

A curated skill library for Qieman design work, organized in the same spirit as anthropics/skills.

## Structure
- `skills/` contains individual skills (one folder per skill).
- Each skill folder includes a required `SKILL.md`.

## Skill Index
- UI/UX: `skills/ui-ux/SKILL.md`
- Motion: `skills/motion/SKILL.md`
- Data Visualization: `skills/data-visualization/SKILL.md`
- Marketing Design: `skills/marketing-design/SKILL.md`
- PPT: `skills/ppt/SKILL.md`
- Xiaogu IP: `skills/xiaogu-ip/SKILL.md`
- AI Workflow: `skills/ai-workflow/SKILL.md`

## Conventions
- Folder names: lowercase kebab-case (e.g., `data-visualization`)
- Required file: `SKILL.md` in each skill folder
- `SKILL.md` begins with YAML frontmatter: `name` and `description`
