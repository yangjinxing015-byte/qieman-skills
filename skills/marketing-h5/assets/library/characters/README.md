# 且慢营销人物与 IP 素材说明

本目录用于存放 **且慢品牌人物、IP、角色类视觉素材**，主要用于营销 H5、品牌传播页、节日活动页、陪伴型专题页等场景。

人物与 IP 素材的核心作用是：

- 传递品牌温度与陪伴感
- 增强页面故事感与互动感
- 提升品牌识别度
- 在合适场景中承载首屏主视觉或内容模块视觉焦点

---

## 素材范围

本目录建议包含以下类型素材：

- 小顾 IP
- 品牌角色延展形象
- 人物插画
- 节日互动人物素材
- 服务陪伴型角色视觉
- 可复用的人物姿态素材

---

## 适用场景

人物与 IP 素材优先用于以下页面：

- 品牌活动页
- 节日营销页
- 情绪型传播页
- 陪伴型专题页
- 品牌故事页
- 服务型产品介绍页

---

## 不建议优先使用的场景

以下页面中，人物素材通常不应作为第一优先主视觉：

- 纯数据总结页
- 年度账单纯数据首页
- 强理性配置说明页
- 以数据 / 图表为绝对主信息的页面

这类页面通常应优先：
- 图标
- 数据意向图形
- 轻量装饰
- 简洁场景辅助图

---

## 使用原则

### 1. 人物服务于主题
人物 / IP 应服务于页面主题，不应喧宾夺主。

### 2. 不抢标题焦点
首屏使用人物素材时，应确保：
- 主标题可读性优先
- CTA 清晰可见
- 人物不压住核心信息区

### 3. 保持品牌气质
人物 / IP 素材应体现且慢品牌的：
- 专业
- 温和
- 可信
- 轻盈
- 有陪伴感

### 4. 避免过度表演化
人物动作、表情与情绪应适度克制，不建议过度夸张、戏剧化或娱乐化。

### 5. 同页风格统一
同一页面中不要混用明显不同风格的人物素材。

## H5 场景示例

### 场景 1：节日营销首屏
推荐使用：
- 小顾 IP
- 节日互动人物素材

使用建议：
- 人物位于右侧或右中区域
- 标题与 CTA 保持左侧主信息区
- 节日氛围可通过 decorations 辅助增强

---

### 场景 2：陪伴型服务专题
推荐使用：
- 小顾 IP
- 服务感人物形象
- 温和互动型角色素材

使用建议：
- 人物用于传递“随时陪伴”“懂你需求”的品牌感受
- 人物动作应自然，不宜过度表演
- 可与少量图标 / 装饰组合使用

---

### 场景 3：品牌故事页
推荐使用：
- 小顾 IP
- 柔和人物场景素材

使用建议：
- 人物与场景共同服务于叙事
- 不抢主文案
- 保持视觉节奏舒展、留白充分

---

### 场景 4：活动专题页
推荐使用：
- 人物主视觉
- 节日角色延展素材
- 活动互动型人物素材

使用建议：
- 人物应增强活动情绪，但不能压住利益点表达
- CTA 与活动时间信息应始终清晰

---

## 推荐命名方式

建议统一使用：

- 小写字母
- kebab-case
- 不使用空格
- 不使用中文文件名
- 必要时补版本号

示例：
- `character-xiaogu-wave-v01.png`
- `character-xiaogu-hold-lantern-v02.png`
- `character-service-companion-v01.png`

---

## 维护原则

- 优先保留品牌识别度高、复用性强的人物素材
- 相同动作或姿态的低质量重复版本应去重
- 若人物风格明显偏离当前品牌体系，应及时归档或清理
- 新增人物素材时应优先考虑：
  - 是否符合且慢品牌调性
  - 是否具备复用价值
  - 是否适合首屏 / 模块 / 节日等明确场景
  - 是否有清晰命名与用途说明

---

## 推荐调用原则

- 陪伴型 / 情绪型 / 节日型页面优先考虑人物素材
- 强金融、强数据页面慎用人物主视觉
- 若使用人物做 Hero 主视觉，需优先保证标题与 CTA 可读性
- HTML Demo 中仅使用本地真实存在的人物素材路径
- 若无匹配人物素材，应明确输出：
  - `缺失素材：人物 / IP（具体用途）`

---

## 不建议的用法

- 不要让人物压住页面主标题
- 不要把人物素材强塞进纯数据页面
- 不要使用与品牌气质不一致的人物风格
- 不要在同一页面混用多个风格明显不同的人物素材
- 不要用默认插画、外部占位角色替代正式 IP / 人物素材
