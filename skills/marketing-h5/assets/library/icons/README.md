# 且慢营销图标素材总说明

本目录用于统一管理 **且慢营销 H5 / 年度账单 / 专题页 / 活动页 / 品牌传播页** 中使用的图标素材。

图标素材主要承担以下职责：

- 辅助信息识别
- 强化数据或功能语义
- 提升页面节奏感与可读性
- 在不破坏金融品牌可信度的前提下增强视觉表达

---

## 目录结构

本目录建议按以下方式组织：

- `cool/`：偏冷静、理性、科技感、金融感的图标
- `warm/`：偏温暖、节日感、活动感、情绪感的图标

---

## 使用原则

### cool/
优先用于：

- 年度账单页
- 数据总结页
- 收益概览页
- 投顾 / 配置 / 分析页
- AI / 智能投顾专题页
- 金融信息导向型活动页

特点：

- 更理性
- 更克制
- 更适合金融科技品牌表达
- 更适合作为数据模块和摘要信息的辅助视觉

### warm/
优先用于：

- 节日营销页
- 活动专题页
- 品牌传播页
- 权益说明页
- 情绪型活动模块

特点：

- 更温暖
- 更有氛围感
- 更适合活动利益点与节日话题表达

---

## 选择优先级

调用图标时建议遵循以下顺序：

1. 用户明确指定的图标精确路径
2. 与当前页面场景最匹配的图标体系（`cool/` 或 `warm/`）
3. 品牌一致性最高、语义最准确的一项
4. 若无匹配项，明确输出：
   - `缺失素材：图标（具体用途）`

---

## 页面场景建议

### 数据 / 金融 / 年度账单类页面
默认优先：
- `cool/`

### 节日 / 活动 / 情绪传播类页面
按需优先：
- `warm/`

### 混合型页面
建议：
- 以一个体系为主，另一个体系为辅
- 不要平均混用两套图标体系
- 保持整页风格统一

---

## 命名建议

建议统一使用：

- 小写字母
- kebab-case
- 不使用空格
- 不使用中文文件名
- 必要时补版本号

示例：
- `icon-cool-bar-chart-growth-v01.png`
- `icon-cool-ai-dashboard-v01.png`
- `icon-warm-festival-gift-v01.png`
- `icon-warm-benefit-card-v01.png`

---

## 维护原则

- 同一语义下避免堆积重复图标
- 新增图标时优先判断：
  - 是否符合且慢品牌调性
  - 是否适合复用
  - 是否与现有体系风格一致
  - 是否具备清晰命名
- 风格不一致、质量偏低或已过时的图标应及时清理或归档

---

## 推荐调用原则

- 同一页面优先保持单一主图标体系
- 数据页面优先调用 `cool/`
- 活动 / 节日页面可按需调用 `warm/`
- HTML Demo 中仅使用本地真实存在的图标路径
- 不要用 emoji、默认图标、占位资源替代正式图标

---
