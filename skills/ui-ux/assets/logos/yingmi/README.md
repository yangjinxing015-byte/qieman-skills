# Yingmi Logos (UI/UX)

Yingmi brand logos used in UI/UX.

## Files (expected)
- `yingmi-logo-regular.(svg|png)`
- `yingmi-logo-white.(svg|png)`

## Usage
- Use `regular` for standard light backgrounds.
- Use `white` for dark/brand backgrounds.

## Notes
- Prefer SVG; use PNG for quick placement only.
