# UI/UX Logos

This folder contains brand logos used in **UI/UX prototypes, documentation, and handoff**.

## Folders
- `qieman/` — Qieman brand logos (regular / white / graphic)
- `yingmi/` — Yingmi brand logos (regular / white)
- `lockups/` — Lockups (slogan / co-brand combinations)

## Naming convention
`<brand>-logo-<variant>.<ext>`

- brand: `qieman` / `yingmi` / `yingmi-qieman`
- variant: `regular` / `white` / `graphic` / `slogan`
- ext: `png` / `svg`

## Notes
- Prefer **SVG** as the source-of-truth when available.
- Keep assets lightweight; avoid large or duplicated binaries.
