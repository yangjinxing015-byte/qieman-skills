# Logo Lockups (UI/UX)

Lockups are combined marks used when layout requires a fixed pairing (slogan / co-brand).

## Files (expected)
- `qieman-logo-slogan.(svg|png)`
- `yingmi-qieman-logo-regular.(svg|png)`
- `yingmi-qieman-logo-white.(svg|png)`

## Usage
- Use lockups only when the design requires the combined mark.
- Prefer `regular` on light backgrounds and `white` on dark backgrounds.

## Notes
- Prefer SVG; use PNG for quick placement only.
- Avoid keeping multiple near-duplicate lockups unless necessary.
