# Marketing Design References

Specs, rules, and reusable writing/design templates for Qieman marketing design.

## What goes here

- Brand and campaign guidelines
- Color usage rules and palettes
- Layout modules for H5 / cards / header hero
- Export specs and QA checklists
- Reusable prompts or structured text templates

## Files

- `h5/color-guidelines_zh.md`: 且慢营销 H5 配色规范
- `h5/card-styles_zh.md`: 卡片样式规范
- `h5/header-hero-guidelines_zh.md`: 首屏 Hero 区规范
- `h5/marketing-design-spec_zh.md`: 营销设计通用规范

## Notes

- `references/` 用于存放 source of truth 类型的规范文档。
- H5 相关文档统一放在 `references/h5/`。
- 不保留与主入口重复的 overview 型文档。
