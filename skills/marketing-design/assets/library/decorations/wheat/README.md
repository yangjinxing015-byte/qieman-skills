# Wheat

Wheat ornaments used in yearly report / data summary layouts.
Typically placed on the left/right side of key numbers or charts.

## What goes here
- Wheat ear flourishes (PNG/SVG), left/right mirrored variants
- Small decorative overlays related to “harvest / growth / annual summary”

## Naming convention
dec-wheat-ear-<left|right>-vNN.<ext>

Examples:
- dec-wheat-ear-left-v01.png
- dec-wheat-ear-right-v01.png

## Notes
- Prefer transparent background PNG (or SVG when available).
- Keep files lightweight; avoid duplicates.
- If you add new styles (gold / gradient / outline), add a style token:
  - dec-wheat-ear-left-gold-v01.png
  - dec-wheat-ear-right-outline-v01.svg
