# Marketing Fonts

This folder contains font files used in **Qieman marketing design** (KV/H5/posters/reports).

## Font roles
- **Alibaba PuHuiTi** (`alibaba-puhuiti/`) — Primary Chinese text
- **MiSans** (`mi-sans/`) — Numbers-only font (KPIs, amounts, returns, chart numbers)

## Folder structure
- `alibaba-puhuiti/`: Alibaba PuHuiTi font files (.ttf/.otf)
- `mi-sans/`: MiSans font files (.ttf/.otf)

## Usage rules (quick)
- Chinese copy: **Alibaba PuHuiTi**
- Digits only: **MiSans** (%, currency, large numbers, chart labels)

See full typography spec:
- `skills/marketing-design/references/typography_zh.md`

## Notes
- Keep file names **clean and consistent** (no Chinese, no spaces).
- Prefer `.ttf` or `.otf`. Avoid duplicates.
- ⚠️ Internal use only. Do not distribute externally unless licensing allows.
