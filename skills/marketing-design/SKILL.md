# Qieman Marketing Design Skill

## Purpose
本 Skill 用于支持且慢营销设计相关内容的生成、整理与规范约束，帮助输出符合且慢品牌调性的营销页面结构、视觉建议、模块说明和可复用的设计描述，便于用于设计执行、策划沟通或 AI 工具生成。

## Scope
本 Skill 适用于以下场景：

- H5 页面结构梳理
- 品牌传播页、活动专题页、节日营销页内容规划
- 首屏 Hero 区设计建议
- 卡片式营销模块设计建议
- 海报 / KV / 活动视觉的结构化描述
- 营销设计相关配色、布局、模块规范整理
- 可直接用于 Figma、即梦、Lovart、H5 搭建工具或 AI 工具的描述词输出

## When to use
当用户需求涉及以下方向时，优先使用本 Skill：

- 需要生成且慢营销 H5 页面结构
- 需要输出活动页、专题页、品牌传播页的设计建议
- 需要统一页面视觉风格、首屏布局、模块节奏或卡片形式
- 需要整理配色规范、模块规范、营销设计说明
- 需要为 AI 生图 / 视频 / 页面生成工具提供结构化提示词
- 需要把营销需求转化为设计可执行的页面框架和视觉规则

## Core principles
处理且慢营销设计相关任务时，遵循以下原则：

1. 保持且慢品牌调性一致，突出专业、可信、轻盈、现代的金融服务感。
2. 页面结构清晰，首屏价值表达明确，重点信息突出，减少无效装饰。
3. 在强调营销传播效果的同时，保留金融品牌应有的专业度与秩序感。
4. 优先采用模块化、卡片化、可复用的表达方式，方便设计系统沉淀与后续复用。
5. 输出内容尽量具体、清晰、结构化，可直接支持设计执行或 AI 生成。
6. 节日、热点、活动场景可以适度增强氛围感，但不能削弱品牌可信度与金融属性。

## Output expectations
输出内容通常应尽量包含以下部分：

- 页面目标
- 目标用户或传播场景
- 页面结构层级
- 模块划分建议
- 首屏 / Hero 区设计建议
- 卡片 / 内容区设计建议
- 配色与视觉风格建议
- 适合设计执行或 AI 工具使用的提示词 / 描述文本

## H5 design principles
在处理 H5 相关需求时，应遵循以下原则：

1. 保持且慢品牌调性一致，突出专业、可信、轻盈、现代的金融服务感。
2. 页面结构清晰，首屏价值明确，重点信息突出。
3. 强调营销表达的同时，保留金融品牌应有的专业度与秩序感。
4. 优先采用模块化、卡片化、可复用的页面表达方式。
5. 输出内容尽量具体、清晰，可直接支持设计执行或 AI 生成。

## Workflow
建议按以下方式处理营销设计任务：

1. 先判断页面业务目标与传播场景
2. 再梳理页面结构与信息层级
3. 结合品牌调性确定整体视觉方向
4. 匹配首屏、卡片、配色、模块等参考规范
5. 输出适合执行和复用的结构化结果

## Default style direction
且慢营销设计默认建议采用以下方向：

- 品牌可信的金融科技感
- 清晰留白与层级分明的结构表达
- 明亮、轻盈、克制的视觉氛围
- 强调关键信息转化路径和模块节奏
- 优先采用卡片化与模块化布局
- 在节日或活动场景中适度增强情绪氛围，但不削弱品牌专业性
